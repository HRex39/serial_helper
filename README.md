# serial_helper
简单的串口助手  
留给需要的人  

## Requirements
· Python3 
· PyQt5  
· PyQt5 - tools  
  
```
pip install pyqt5
pip install pyqt5-tools
```  
## How to use
#### Windows
```
python main.py
```
#### Linux/Mac
```
python3 main.py
```
